# Lightness Gym - Ladies Fitness in Kuwait

A modern, premium website for Lightness Gym — Kuwait's premier ladies-only fitness destination.

Built with **Next.js**, **TypeScript**, **Tailwind CSS**, and **Framer Motion**.

## Getting Started

### Prerequisites

- Node.js 18+ installed
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

### Deploy to Vercel

This project is ready to deploy on [Vercel](https://vercel.com). Simply connect your GitHub repository and deploy.

## Customization

### Replace Placeholder Images

Search for `{/* REPLACE: Use actual gym photo */}` comments throughout the components to find all placeholder Unsplash images that should be replaced with actual gym photos.

### Replace Logo

Search for `{/* REPLACE: Add actual logo image here */}` to find the logo placeholder in the Navbar and Footer.

## Tech Stack

- **Next.js 15** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** for animations
- Fully responsive, mobile-first design
- SEO optimized with Open Graph meta tags
